# SafePlay Anti-Cheat

SafePlay Anti-Cheat is a powerful security module for Roblox games.  
It protects your game from common and advanced exploits, including speed hacks, remote abuse, and suspicious behavior. The system is lightweight, optimized, and configurable to fit any game type.

## Features
- Secures remotes and events
- Detects speed and movement exploits
- Monitors suspicious player behavior
- Configurable checks for flexibility
- Lightweight and scalable
- Actively maintained and expandable
